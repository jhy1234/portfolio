Colour Correction & Non-Destructive Photo Editing

→ In this section, I will demonstrate my knowledge in creating a detailed tutorial regarding photo editing, espacially in color correction.

01. Check the file (Size, Resolution, Colour Space)

First of all, it is important to check if the file I'm working with meets the appropriate specifications for its intended use.

After checking the image size and resolution, I will check the colour profile in Edit > Convert to Profile.

02. Non-destructive work flow

Now that I'm sure that I'm working on the right file, I need to decide what needs to be worked on.

  • Color correction
  • Retouching
  • Enhancement

Non-destructive editing allows me to make changes without overwriting the original image data, which remains available in case I want to revert to it. Furthermore, non-destructive editing doesn’t remove data from the image, so the image quality doesn’t degrade when I make edits. This allows for more refined adjustments in Photoshop and therefor can produce a higher quality image.

03. Color Correction

Ideally, I should use color checkers before taking the photos to correct the colour accurately. Here I would do it manually. The decisions are personal, and will vary for different pircures.

Black and White Adjustments

The first step to correct the color is to adjust the black and white points by using a levels adustment layer.

The black point should be set in the darkest point of the neutral shadow but still hold detail.

The next step is to continue adjusting the black and white areas and try to reveal more information in the shadows and rescue information in the highlights. Unfortunately, there isn't a shadows and highlights adjustment layer available in Photoshop, so I use: IMAGE > ADJUSTMENTS > SHADOWS/HIGHLIGHT. To achieve this while still remaining non-destructive workflow, I duplicate the background layer to isolate the adjustments I'm about to make onto a different layer. Then rename the new layer to SHADOWS + HIGHLIGHTS to keep the layers organized.

Correcting the colour

First of all, I use the threshold adjustment layer and color sampler tool to find black, white and grey sample points.

For black and white points, the process is similar to black and white adjustments.

For the neutral midtone points of the image, I create a new blank layer underneath the threshold adjustment layer. Fill this new blank layer with 50% neutral grey. Then I change the layer blending mode option to difference on the new layer.
It is now showing where the neutral midtone points of the image are located. Move the slider to the left and slowly start to move back to the right. Use the color sampler tool to set an indicator in a small cluster of pixels on a part of the image that I think should be grey.

There are now 3 indicator points set, representing the shadows, midtones and highlights of the image.

Add a Curves Adjustment Layer to the image. Double click on the black eyedropper tool or “sample in image to set black point” tool. Locate the previously set indicator points for black, grey and white, using the eyedropper tools in the curves adjustment layer, zoom in and click on the black point with the black eyedropper tool, grey point with the grey eyedropper tool and white point with the white eyedropper tool.

And now this image's colour has been corrected.

04. Enhance Colour and Detail

I use hue/saturation adjustment layer to slightly change the color and create a more engaging photograph.

Then I use unsharpen mask to enhance the details. The filter is not available as an adjustment layer, so I need to duplicate the last used image layer, and apply the filter to the duplicated layer.

05. Retouching

On top of the layer stack, I create a new blank layer to complete the retouching processes. In this way all I do maintains non-distructive. I use the spot healing brush, healing brush,patch tool, and clone stamp tool to remove unwanted elements.
